STRUCTURE

Framework Modules

Framework organization designed for scalability, maintainability and reusable automation components.

⚙️

core/config

Centralized framework configuration management including environment settings and execution properties.

🛠️

core/utils

Reusable utility classes used across the framework to reduce code duplication and improve maintainability.

🚗

ui/drivers

Contains DriverFactory implementation responsible for browser initialization and WebDriver lifecycle management.

🏗️

ui/base

BasePage, BaseTest and AuthenticatedBaseTest providing common framework functionality and reusable actions.

📄

ui/pages

Page Object Model implementation containing page-specific locators and business actions.

🧪

ui/tests

TestNG test classes validating application workflows through reusable page objects.
